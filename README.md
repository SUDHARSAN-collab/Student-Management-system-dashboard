🎓 Student Management System Dashboard
📌 Project Overview

The Student Management System Dashboard is a data visualization project that helps educational institutions monitor and analyze student-related data efficiently. The dashboard provides insights into student enrollment, attendance, academic performance, and department distribution using interactive visualizations.

This project transforms raw student data into meaningful insights that help administrators and faculty make better academic and management decisions.

🎯 Project Objectives

Track total student enrollment

Analyze department-wise student distribution

Monitor student academic performance

Identify attendance patterns

Provide a clear overview of student statistics

🛠️ Tools & Technologies Used

Power BI – Dashboard development and visualization

Microsoft Excel / CSV – Data storage and preprocessing

DAX (Data Analysis Expressions) – Calculations and measures

Data Modeling – Relationship building between tables

📂 Dataset Description
Column Name	Description

Student_Name	Name of the student
Gender	Male / Female
Department	Department name
Year	Academic year
Attendance_Percentage	Student attendance rate
Marks	Marks obtained
Result	Pass / Fail
